\# Pristine Cleaning App (Android)



Pristine Cleaning App is a native Android application designed to connect \*\*customers\*\* with \*\*professional cleaners\*\*.  

The system allows customers to post cleaning jobs and cleaners to apply, manage, and complete them.



This project was developed as part of my Mobile Application Development learning journey.



\---



\## Key Features



\### Customer Side

\- Register and login

\- Create and post cleaning jobs

\- View job queue and job status

\- Review cleaners after job completion

\- View cleaner reviews



\### Cleaner Side

\- Register and login

\- Browse available jobs

\- Apply for jobs

\- View accepted jobs

\- Update job status

\- Review customers



\### Review System

\- Two-way rating system:

&#x20; - Customers rate cleaners

&#x20; - Cleaners rate customers

\##  Database Entities



\- Customer

\- Cleaner

\- Job

\- Review



The app uses a local SQLite database managed via `DatabaseHelper.java`.



\---



\## App Architecture



The application follows a \*\*multi-activity architecture\*\* with separate dashboards for customers and cleaners.



Main modules include:

\- Authentication module

\- Job management module

\- Review \& rating module

\- SQLite database layer
